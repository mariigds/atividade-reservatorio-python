# Controle de Níveis de Reservatório 💧

Este projeto consiste em um sistema de monitoramento desenvolvido em Python que fornece alertas visuais sobre os níveis de água de um reservatório. Ele utiliza a biblioteca Colorama para facilitar a identificação visual de estados críticos diretamente no terminal.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python
* **Biblioteca:** Colorama (estilização do terminal)
* **Versionamento:** Git e GitHub

## 🛠️ Funcionalidades e Níveis de Alerta
O sistema monitora 5 estados distintos para garantir a segurança do reservatório:

1.  **Nível Crítico (Vermelho):** Indica volume perigosamente baixo.
2.  **Nível Baixo (Amarelo):** Alerta de atenção para consumo reduzido.
3.  **Nível Médio (Verde):** Volume seguro e estável.
4.  **Nível Alto (Ciano):** Estado de transição do volume para Alto.
5.  **Nível Pleno (Azul):** Reservatório em sua capacidade máxima operacional.
